Rejoin a game you left.

#### How does it work?
When you leave a game or if you get disconnected you can re-join it, based on some conditions.

#### Usage

| Command    | Description    |
|------------|----------------|
| /rejoin    | re-join a game |
| /bw rejoin | re-join a game |
